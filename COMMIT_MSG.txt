Major Feature Release: Camera, Microphone, RTC, Presence Sensor, Maps Overhaul

=== NEW SENSOR MODULES ===

i2csensor-ds3231.cpp/h: NEW - DS3231 RTC clock sensor driver
  - Full I2C driver for DS3231 precision real-time clock
  - Time sync with NTP, manual time setting support
  - Alarm functionality and temperature reading
  - Settings module with auto-start and time-set tracking

i2csensor-ds3231-oled.h: NEW - RTC OLED display mode
  - Date/time display with formatting options
  - Temperature readout from DS3231

i2csensor-ds3231-web.h: NEW - RTC web interface card
  - Live time display and manual time setting
  - NTP sync status and controls

i2csensor-sths34pf80.cpp/h: NEW - STHS34PF80 IR presence/motion sensor driver
  - Full I2C driver for ST IR presence detector
  - Ambient temperature, presence detection, motion detection
  - Configurable polling interval and thresholds
  - Low-priority I2C access with mutex yielding to gamepad

i2csensor-sths34pf80-oled.h: NEW - Presence sensor OLED display mode
  - Live ambient/presence/motion value display
  - Visual indicators for detection states

i2csensor-sths34pf80-web.h: NEW - Presence sensor web interface card
  - Live streaming of presence data
  - Fixed element IDs for proper data binding
